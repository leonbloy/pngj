Class PngReaderInt

  • java.lang.Object
    • ar.com.hjg.pngj.PngReader
      • ar.com.hjg.pngj.PngReaderInt

  • public class PngReaderInt
extends PngReader
    Trivial extension of PngReader that uses ImageLineInt.

    In the current implementation this is quite dummy/redundant, because (for backward compatibility) PngReader already uses a ImageLineInt.

    The factory is set at construction time. Remember that this could still be changed at runtime.
