Class ChunkReader

  • java.lang.Object
    • ar.com.hjg.pngj.ChunkReader
  • Direct Known Subclasses:
    DeflatedChunkReader

    public abstract class ChunkReader
extends Object
    Parses a PNG chunk, consuming bytes in one mode: ChunkReader.ChunkReaderMode.BUFFER, ChunkReader.ChunkReaderMode.PROCESS, ChunkReader.ChunkReaderMode.SKIP.

    It calls chunkDone() when done. Also calls #processData(byte[], int, int) if PROCESS mode. Apart from thas, it's totally agnostic (it doesn't know about IDAT chunks, or PNG general structure)

    The object wraps a ChunkRaw instance (content filled only if BUFFER mode); it should be short lived (one instance created for each chunk, and discarded after reading), but the wrapped chunkRaw can be (usually is) long lived.
